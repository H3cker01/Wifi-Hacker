# ⚡ WIFI HACKER ⚡
Developed by: Hecker
A professional-grade wireless security auditing tool with a synchronized hardware-check engine and high-contrast hacker aesthetic. Now updated to a Standalone Executable (.exe)—no Python installation required!
## 🚀 Features
Zero-Dependency: Runs on any Windows 10/11 machine without installing Python or libraries.
Deep Sync Technology: Hecker's custom logic to prevent skipping the correct password.
Colorama Effects: High-contrast, multi-color terminal interface.
Typewriter Warning: Dramatic character-by-character success messages.
Auto-Logging: Cracked keys are automatically saved to cracked_passwords.txt.
## 💻 Requirements
Windows 10 or later
Administrative Privileges (Required to access WiFi hardware)
Wordlist: e.g., rockyou.txt or the included probable.txt
## 🛠️ Installation & Setup
Download the repository or just the wifi.exe from this repository.
Place your wordlist (e.g., probable.txt) in the same folder as the .exe.
No Python or Pip needed! All dependencies are bundled inside the file.
## 🕹️ Usage
Right-click wifi.exe and select "Run as Administrator".
Press ENTER to start the live WiFi scanner.
Press Ctrl+C once your target appears, then enter its ID number.
Enter the path to your wordlist (e.g., probable.txt).
Sit back and watch the injector work.
once the key is found it is located at cracked_paswords.txt in same folder as wifi.exe
